Projekte einer bestimmten Größe bedürfen einer guten Architektur. Eine gute Planung ist ein entscheidender Faktor für eine geschmeidige Projektentwicklung. Es ermöglicht Wachstum und Skalierbarkeit ohne die Produktivität zu verschlechtern, das Projekt bleibt beherrschbar.

Marble, I perceive, covers a multitude of sins. Aldous Huxley

So ist es auch mit Quellcode! In der Web-Entwicklung kommen in einem System oft eine ganze Reihe verschiedener Technologien zum Einsatz. Eine lose Kopplung der Teilsystem sorgt dafür, dass Komponenten unabhängig von einander austauschbar sind und wachsen können. Anstelle monolithischer Konstrukte optimieren wir unsere Projekte als Zusammenspiel spezialisierter Komponenten.

Write programs that do one thing and do it well. Write programs to work together. Write programs to handle text streams, because that is a universal interface. Doug McIlroy

Simple text streams can be replaced e.g. by message-queues where need be, but the underlaying principle is as true nowadays as it has been in the 70s during the early phases of UNIX.
